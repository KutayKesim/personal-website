Personal Website Project

Creating a HTML file in Atom

1. Create Folder (PC)
2. Folder again (inside Atom)
3. Create file as /index.html (conventional for home page)
4. Add boilerplate code i.e. templates, generic start codes
for HTML just type "html" (as we have "emmet" plugin installed)

Meta tags

They contain key information esp. for search engines such as:
charset="UTF-8">
name="description" content="Free Web tutorials"> This is what will be displayed in search engines
name="keywords" content="HTML, CSS, JavaScript"> This is for SEO
name="author" content="John Doe">
name="viewport" content="width=device-width, initial-scale=1.0">

NOTE You can check other websites keywords (?) and descriptions by viewing their page source

HTML Elements

We can use the shortcuts due to our package

h1 for heading 1
p for a paragraph

NOTE
For bold use strong over b
For italic use em over i

Lists

  • ul for the unordered list i.e. bullet point
  • li for list item
  • ol for ordered (1,2,3 etc.)
  • for other orderings use type attributes e.g. type="i" for roman numerals

Image Elements

To add an image the tag "img" is used. You also need to specify the source and that's the source of the image that will be displayed (img src="abc.png"). That can either be a URL so it can fetch it from the internet, or it can be a local image that's included in the same directory as your website.
